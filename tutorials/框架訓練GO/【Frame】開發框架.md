# 【Frame】開發框架

| FastAPI | FastAPI 是一個基於Python 的現代化、高效能的Web 框架，專為構建API (應用程式程式設計介面) 而設計。 | [🔗 點我](https://fastapi.tiangolo.com/) |
| --- | --- | --- |
| Django | Django 是一個用Python 語言編寫的開源Web 應用程式框架(framework)。 它主要的功能是幫助開發者快速且有效地建立網站和Web 應用程式。 Django 提供了許多內建的功能和組件，可以簡化開發流程，例如使用者驗證、資料庫操作、URL 路由、模板引擎等等，讓開發者可以專注於網站的業務邏輯，而不用從頭開始編寫這些基礎功能。 | [🔗 點我](https://www.djangoproject.com/) |
| Flask | Flask 是一個用 Python 編寫的輕量級 Web 應用程式框架。它也被稱為微框架，因為它提供了核心功能，如路由和模板渲染，但沒有強制使用特定的工具或庫，例如資料庫或表單驗證。 Flask 允許開發者根據需要添加功能，例如使用擴充功能來整合資料庫、驗證表單等。 | [🔗 點我](https://flask.palletsprojects.com/en/stable/) |
