# 資安名詞整理

### 資安名詞整理

> 精簡、合併重複條目，按主題分區。每區以「縮寫/名詞｜全稱｜重點說明」呈現。
> 

---

## I. 安全標準與組織

| 縮寫 / 名稱 | 全稱 (英文) | 重點說明 |
| --- | --- | --- |
| CVE | Common Vulnerabilities and Exposures | 已知漏洞的標準字典，為每個漏洞分配唯一 ID。 |
| CVSS | Common Vulnerability Scoring System | 漏洞嚴重度量化評分 (0–10)，用於修補優先級。 |
| MITRE | MITRE Corporation | CVE 的創建者；維護 ATT&CK 等框架。 |
| FIRST | Forum of Incident Response and Security Teams | CVSS 的官方維護與管理。 |
| SANS | SANS Institute | 安全培訓與認證；提供 ISC、NewsBites 等資源。 |
| DHS | Department of Homeland Security | 提供威脅情報與 AIS 自動指標共享服務。 |
| Cisco Talos | Threat Intelligence Group | 商業威脅情報與保護服務。 |

## II. 防禦、加密與存取控制

| 縮寫 / 名稱 | 全稱 (英文) | 重點說明 |
| --- | --- | --- |
| IPS | Intrusion Prevention System | 主動阻擋惡意行為或攻擊。 |
| IDS | Intrusion Detection System | 偵測潛在威脅並告警。 |
| NAC | Network Access Control | 設備合規性檢查與網路存取控管。 |
| VPN | Virtual Private Network | 建立加密隧道，支援遠端安全連線。 |
| IPsec | IP Security | 網路層加密、完整性與身分驗證。 |
| AES | Advanced Encryption Standard | 常用對稱分組加密標準。 |
| BitLocker | - | Windows 內建全磁碟/卷加密。 |
| EFS | Encrypting File System | Windows NTFS 檔案/資料夾級加密。 |
| ACL | Access Control List | 資源權限規則清單。 |
| DAC | Discretionary Access Control | 由資源擁有者決定權限的模型。 |

## III. 網路基礎設施與常見埠口

| 名詞 | 全稱 (英文) | 重點說明 |
| --- | --- | --- |
| NAT | Network Address Translation | 私有 IP 轉公網 IP。 |
| Proxy | Proxy Server | 中繼、快取、過濾與存取控制。 |
| QoS | Quality of Service | 流量優先級與網路效能保障。 |
| VLAN | Virtual LAN | 邏輯網段隔離。 |
| GRE | Generic Routing Encapsulation | 封裝隧道，無加密。 |
| Port 22 | SSH | 遠端加密登入與命令執行。 |
| Port 23 | Telnet | 明文遠端登入，不安全。 |
| Port 20/21 | FTP | 21 控制，20 數據，皆明文。 |

## VI. Netstat 常用參數（Windows）

| 參數 | 說明 |
| --- | --- |
| -o | 顯示 PID。 |
| -a | 顯示所有連線與偵聽埠。 |
| -n | 以數字顯示 IP 與埠。 |
| -r | 顯示路由表。 |
| -s | 顯示各協議統計。 |
| -p protocol | 篩選指定協議，如 tcp 或 udp。 |

## VII. NSLookup 模式與常用設定

| 項目 | 說明 |
| --- | --- |
| 非交互模式 | 單次查詢後退出。 |
| 交互模式 | 可連續執行多個查詢。 |
| set recurse | 啟用遞迴查詢。 |
| set querytype | 切換 RR 類型，如 A、MX 等。 |
| set retry | 設定未回應時的重試次數。 |
| set domain | 設定預設 DNS 網域。 |

| 虛擬機 (VM) | 虛擬化硬體 (創造獨立作業系統環境) | 高 (獨立 OS，軟硬體完全隔離) | 高（包含完整 OS），啟動較慢 |
| --- | --- | --- | --- |
| 容器 (Container) | 虛擬化應用程式 (打包應用程式與依賴項) | 中 (共享宿主機核心，由 OS 核心隔離) | 低（不含 OS），啟動極快 |
| 虛擬機監控程式 (Hypervisor) | 管理與分配資源 (讓多個 OS 能在單機運行) | 最高 (控制所有 VM 的核心層) | 屬於基礎架構，資源消耗相對固定 |
