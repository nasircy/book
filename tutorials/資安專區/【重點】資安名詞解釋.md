# 【重點】資安名詞解釋

### Netstat 命令參數解釋

| **命令** | **參數功能** | **實際輸出內容** |
| --- | --- | --- |
| `netstat -o` | 顯示程序的 ID（PID） | 列出所有活動的 TCP 與 UDP 連線，並在最後一欄顯示各連線所屬的程序 ID（PID）。 |
| `netstat -a` | 顯示所有連線與偵聽埠 | 列出所有活動的 TCP 連線與目前偵聽中的 TCP、UDP 埠口。 |
| `netstat -r` | 顯示路由表 | 顯示 IP 路由表內容，用於了解封包如何由本機傳送至目的地。 |
| `netstat -e` | 顯示乙太網路統計 | 顯示網路介面統計（收發位元組、單播與非單播封包、錯誤與丟棄等）。 |

> 小提示：實務上常搭配 `netstat -ano`（顯示位址、連線狀態與 PID），再用 `tasklist /fi "PID eq <數字>"` 或工作管理員比對。
>

### 資訊安全管理與 Linux 權限

| **概念/命令** | **類型** | **目的與解釋** |
| --- | --- | --- |
| 變更管理目標 1 | 預防風險 | 在變更前進行安全影響分析，避免對安全造成負面影響。 |
| 變更管理目標 2 | 流程控制 | 安裝或修改系統時更新相關文件，利於事件響應與稽核。 |
| 安全控制 | 預防性控制 | 阻止事件發生。如：最終用戶安全培訓以減少失誤。 |
| 安全控制 | 糾正性控制 | 修復或恢復事件造成的損害。如：業務連續性計畫。 |
| Linux `chmod` | 權限命令 | 以 `u, g, o, a` 與 `+ - =` 調整檔案/目錄存取權限。 |
| 正確 `chmod` 範例 | `chmod og-wx file` | 移除群組（g）與其他（o）的寫入（w）與執行（x）權限。 |

### Netstat 與 Windows 稽核對照

| **命令/策略** | **功能** | **稽核目的** |
| --- | --- | --- |
| `netstat -o` | 顯示所有連線及對應 PID | 追蹤是哪個應用程式使用哪個連線。 |
| `netstat -a` | 顯示所有活動連線與偵聽埠 | 總覽網路活動，但不含應用程式名稱。 |
| `netstat -p <protocol>` | 篩選指定協議（如 TCP 或 UDP） | 僅檢視特定類型的連線。 |
| 稽核：物件存取 | Windows 稽核策略 | 追蹤對檔案、資料夾等物件的讀寫刪除行為。 |
| 稽核：登入 | Windows 稽核策略 | 記錄使用者登入與登出事件。 |
