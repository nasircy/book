# 【Mac】完整清除指令

| **類型** | **主要路徑 (點擊「前往檔案夾」輸入)** | **風險等級 & 說明** |
| --- | --- | --- |
| **個人快取** | `~/Library/Caches` | **中風險**
這是最主要的垃圾來源。建議**刪除裡面的子檔案夾**，而不是刪除 Caches 檔案夾本身。 |
| **App 支援檔** | `~/Library/Application Support` | **高風險**
當**確定已刪除某 App** 後，才來此刪除同名的殘留檔案夾。**切勿**刪除還在使用的 App 資料！ |
| **App 偏好設定** | `~/Library/Preferences` | **高風險**
存放 App 的設定檔。同樣只建議刪除**已移除 App** 的 `.plist` 殘留檔。 |
| **個人日誌** | `~/Library/Logs` | **低風險**
App 和系統的運作紀錄，通常可以安全刪除，但空間佔用不大。 |
| **下載項目** | `~/Downloads` | **安全**
您自己下載的檔案，請手動檢查並刪除不再需要的內容。 |
| **iOS 備份** | `~/Library/Application Support/MobileSync/Backup` | **中風險**
佔空間大戶。**強烈建議**使用 Mac 內建的「儲存空間管理」功能來刪除，而非手動刪除。 |
| **系統快取** | `/Library/Caches` | **極高風險 (不建議)**
這是系統層級的快取。**新手請勿觸碰**。 |
| **系統日誌** | `/Library/Logs` (或 `/var/log`) | **高風險 (不建議)**
系統層級的日誌。**新手請勿觸碰**。 |
